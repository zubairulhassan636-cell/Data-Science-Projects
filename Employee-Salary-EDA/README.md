# Employee Salary Dataset - Advanced EDA & Feature Engineering

## 📌 Project Overview

This project focuses on **Advanced Exploratory Data Analysis (EDA)** and **Feature Engineering** using the Employee Salary Dataset. The objective is to clean and transform raw data into a high-quality dataset that is ready for Machine Learning models.

The project follows the Data Science Project 1 guidelines, including missing value handling, outlier treatment, statistical analysis, feature engineering, and data visualization.

---

## 🎯 Project Objectives

* Load and inspect the dataset.
* Perform Exploratory Data Analysis (EDA).
* Handle missing values using statistical imputation techniques.
* Detect and treat outliers using the Interquartile Range (IQR) method.
* Create new meaningful features from existing data.
* Visualize important patterns and relationships.
* Prepare a clean dataset for future Machine Learning applications.
